<h1 align="center">宿院二手短租平台</h1>

## 需求

### 用户模块
- 用户登录注册，普通手机号注册（模拟）
- 修改个人信息

### 商品模块
- 首页显示最新发布物品 redis
- 根据物品的分类查看所有物品
- 查看商品详情
- 加入商品关注
- 查看我的商品关注
- 发布商品 vsftpd
- 查看我发布的商品
- 搜索商品 solr
